# Interactive Chemistry Energy Profiles & Kinetics Simulator

An interactive, web-based educational tool designed to visualize chemical thermodynamics and kinetics. This simulator allows students and educators to explore energy profiles, activation energy, catalysis, and Maxwell-Boltzmann distributions in real-time.

## Features
* **Interactive Sliders:** Manually drag the reaction forward and backward to see real-time updates to Potential Energy (PE), Kinetic Energy (KE), and Temperature.
* **Exothermic vs. Endothermic:** Clear visual comparisons of energy absorption vs. energy release.
* **Catalysis Race:** Watch a catalyzed reaction physically race an uncatalyzed reaction, proving how lowering Activation Energy ($E_a$) increases reaction speed.
* **Collision Theory (Maxwell-Boltzmann):** A split-screen simulation showing how heating a system pushes a larger percentage of particles past the Activation Energy threshold, complete with a live particle container.

## How to Use
This project requires no installation, build tools, or dependencies. 
1. Download or clone this repository.
2. Open the `index.html` file directly in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Use the slider to control the reaction or press the "Play" button to watch it animate.
4. Click the "Next" button at the bottom to cycle through the four different educational modules.

## Tech Stack
* **HTML5 Canvas:** Used for all high-performance rendering and animations.
* **Vanilla JavaScript:** Handles the physics, easing functions, and DOM manipulation.
* **CSS3:** For styling, layout, and responsive dashboard design.
